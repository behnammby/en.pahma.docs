# What is this documentation | این مستندات در مورد چیست

This is the official documentation of Afaq Artificial Intelligence Platform (Pahma).

Pahma helps you create your own softbot quickly and easily. You can create any flows you need and then connect it to your messenger of choise.

Follow our documentation to know more.
